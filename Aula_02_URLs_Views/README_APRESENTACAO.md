# 📊 Apresentação Aula 02 - URLs e Views

## 🎯 Arquivo Criado

**Aula_02_Apresentacao.html** - Apresentação interativa em HTML usando Reveal.js

## 🚀 Como Usar

### Opção 1: Abrir Diretamente no Navegador

1. Navegue até a pasta `Aula_02_URLs_Views`
2. Clique duas vezes em `Aula_02_Apresentacao.html`
3. O arquivo será aberto no seu navegador padrão

### Opção 2: Via Terminal/Servidor Local

```bash
# Navegue até a pasta
cd Aula_02_URLs_Views

# Inicie um servidor HTTP simples (Python 3)
python3 -m http.server 8000

# Acesse no navegador:
# http://localhost:8000/Aula_02_Apresentacao.html
```

## 💡 Dicas para Visualização Perfeita

1. **Tela Cheia**: Pressione `F` para modo tela cheia
2. **Zoom do Navegador**: Se o conteúdo parecer pequeno, use Ctrl/Cmd + '+' para aumentar
3. **Visão Geral**: Pressione `Esc` para ver todos os slides e navegar rapidamente
4. **Resolução**: A apresentação é otimizada para 1280x720 (16:9)

## 🔧 Personalização

Para modificar a apresentação, edite o arquivo HTML:

- **Tema**: Linha 9 - Altere `white.css` para outros temas disponíveis
  
- **Cores**: Seção `<style>` - Altere as cores:
  - `#0C4B33` - Verde IFRN
  - `#C8102E` - Vermelho dos banners
  
- **Tamanho de Fonte**: Edite a propriedade `font-size` no estilo `.reveal`

- **Conteúdo**: Edite as seções `<section>` para modificar slides
