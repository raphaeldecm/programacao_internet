🖼️ O que são Static Files?

Arquivos que são entregues ao navegador "como estão", sem processamento pelo Python/Django.

  • Folhas de Estilo: Arquivos .css.
  • Imagens: .jpg, .png, .svg.
  • Scripts: Arquivos .js (lado do cliente).

No Django, gerenciamos isso via app django.contrib.staticfiles.

📁 Organização das Pastas

Assim como nos templates, usamos Namespaces para evitar conflitos entre apps.

meu_app/
└── static/
    └── meu_app/ <-- Namespace do App
        ├── css/
        │   └── style.css
        └── img/
            └── logo.png

⚡ A Tag {% load static %}

Para usar arquivos estáticos, precisamos avisar o motor de templates do Django no topo do arquivo.

templates/meu_app/home.html

            {% load static %}
            <!DOCTYPE html>
            <html>
            <head>
                <link rel="stylesheet" href="{% static 'meu_app/css/style.css' %}">
            </head>
            <body>
                <img src="{% static 'meu_app/img/banner.jpg' %}" alt="Banner">
            </body>
            </html>

⚙️ Bastidores: settings.py

O Django precisa saber onde procurar esses arquivos. Por padrão, ele busca na pasta static de cada app registrado.

settings.py

            # URL que o navegador usará para acessar os arquivos
            STATIC_URL = 'static/'

            # Onde o Django buscará arquivos extras (fora dos apps)
            STATICFILES_DIRS = [
                BASE_DIR / "static",
            ]

🏆 Boas Práticas: CSS no Django

  • Reset CSS: Use sempre um reset para garantir consistência entre navegadores.
  • Caminhos Relativos: Dentro do seu CSS, caminhos para imagens devem ser relativos ao arquivo .css.
  • Foco em UX: Já que estamos em uma abordagem Visual-First, incentive o uso de variáveis CSS para cores do IFRN.
Dica: Em produção, usaremos o comando collectstatic, mas em desenvolvimento (DEBUG=True), o Django cuida de tudo sozinho.

🛠️ Desafio: O Site com "Estilo"

  1. No seu app, crie a pasta static/nome_do_app/css/.
  2. Crie um arquivo style.css e defina uma cor de fundo e uma fonte para o body.
  3. No seu template base.html, carregue o CSS usando a tag {% static %}.
  4. Desafio Extra: Adicione uma imagem na pasta img/ e exiba-a na sua página inicial.

⏱️ Tempo: 40 min | 💡 Verificação: O console do navegador (F12) mostra erro 404 se o caminho estiver errado!
