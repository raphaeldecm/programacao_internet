🚀 Tutorial Completo de Configuração

Passo 01Ambiente Virtual

Isolando as dependências do projeto Mercado.


            # Criar a venv
            python -m venv venv

            # Ativar (Windows)
            venv\Scripts\activate
            
            # Ativar (Linux/Mac)
            source venv/bin/activate

            # Instalar Django
            pip install django

Passo 02Projeto e App

Iniciando a "Nave Mãe" (Projeto) e o Módulo (App).


            # Criar o projeto mercado
            django-admin startproject mercado .

            # Criar o app produtos
            python manage.py startapp produtos
Nota: O ponto final no final do startproject evita que o Django crie uma pasta extra com o mesmo nome.

Passo 03Settings: Apps e Templates

Avisando ao Django onde tudo está.

mercado/settings.py

            INSTALLED_APPS = [
                ...,
                'produtos',
            ]

            TEMPLATES = [
                {
                    ...,
                    # Pasta global para a Home
                    'DIRS': [BASE_DIR / 'templates'], 
                    ...,
                },
            ]

Passo 04Settings: Estáticos

Configurando a busca manual de arquivos CSS/IMG.

mercado/settings.py (Final do arquivo)

            STATIC_URL = 'static/'

            # CRIE ESTA VARIÁVEL (Não vem por padrão):
            STATICFILES_DIRS = [
                BASE_DIR / "static",
            ]

Passo 05A View da Home

Criando a primeira ponte visual no arquivo central.

mercado/urls.py

            from django.contrib import admin
            from django.urls import path, include
            from django.shortcuts import render

            # View simples para feedback rápido
            def home_view(request):
                return render(request, 'home.html')

            urlpatterns = [
                path('admin/', admin.site.urls),
                path('', home_view, name='home'),
                path('produtos/', include('produtos.urls')),
            ]

Passo 06Desacoplamento do App

Criando as regras internas do app produtos.

produtos/urls.py (Crie este arquivo)

            from django.urls import path
            from . import views

            app_name = 'produtos'

            urlpatterns = [
                path('lista/', views.lista, name='lista_produtos'),
            ]
produtos/views.py

            def lista(request):
                return render(request, 'produtos/lista.html')

📁 Hierarquia de Pastas

Como o seu projeto deve parecer no VS Code:

mercado/
├── static/ <-- Global
├── templates/
│   └── home.html
├── produtos/
│   ├── static/produtos/css/style.css
│   └── templates/produtos/lista.html
└── manage.py

🚀 Hora do Teste!

No terminal, execute:

python manage.py runserver

Acesse no navegador:

  • Home: 127.0.0.1:8000/
  • App: 127.0.0.1:8000/produtos/lista/

Dica: Se a página carregar, a arquitetura está correta!
